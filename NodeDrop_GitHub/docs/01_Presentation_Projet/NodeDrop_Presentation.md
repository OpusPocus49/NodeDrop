# NodeDrop — Présentation du projet

## Nom du logiciel

NodeDrop

## Description

NodeDrop est une application portable permettant de transférer facilement des fichiers et dossiers entre plusieurs ordinateurs.

La première version est conçue pour fonctionner sur un réseau local (LAN) et permet la découverte automatique des machines exécutant l'application.

Les utilisateurs peuvent établir une connexion authentifiée entre deux machines et transférer des fichiers avec suivi de progression.

L'application est conçue avec une architecture modulaire afin de permettre une évolution future vers un mode de transfert distant entre réseaux différents.

## Objectif principal

Permettre un transfert simple, rapide et fiable de fichiers volumineux entre machines appartenant à une même organisation ou un même environnement privé.

## Objectifs secondaires

- découverte automatique des machines sur le réseau
- interface simple et intuitive
- validation des connexions par l’utilisateur
- transfert de fichiers et dossiers
- suivi des transferts en temps réel
- architecture extensible

## Public cible

- usage personnel
- petites équipes
- utilisateurs souhaitant échanger des fichiers entre machines sans configuration réseau complexe
